# Portfolio

This is my personal portfolio website built with Next.js, React and TypeScript. It showcases my skills, commercial experience in building dealer websites, and selected pet projects. The site supports three languages (UA / EN / RU), has dark/light themes, responsive layout and basic SEO optimisations (metadata, Open Graph, sitemap, JSON-LD).
